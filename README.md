## Game
I decided to make a small application inspired by how I imagined the Harry Potter game you mentioned would work with gesture recognition. Instead of magic lessons, I however decided to do an application where you have to defeat monsters. The monsters are some of my plushies that I took a photo of and then pixelated, because I though it would give it a fun and unique element.
## Game Logic
- the player has to defeat 6 monsters within 60 seconds
- monsters are defeated after 3 gestures are performed correctly
- gestures have to be drawn in the lower part underneath the monsters, it works best if you draw right on top of the gesture you have to perform
## Other Controls
- start the game with 'P'
- restart the game with 'S'
- quit game with 'Q'
